Remote Light Control System Using Netpie

This project aimed to create a remote-controlled lighting system via smartphone using NETPIE, allowing users to control lights from anywhere. It enhances convenience, reduces unnecessary energy consumption, and improves home safety. NETPIE is a cloud-based IoT platform that serves as a bridge, enabling IoT devices to exchange data with each other or with mobile and web applications quickly and securely. By connecting a mobile internet to the board, users can remotely turn lights on and off through the NETPIE platform without having to set up their own server.

Gas Leakage Detection and Alarm System

This project aimed to develop a gas leak warning system. The gas sensor sends data to an ESP board, which displays the detected gas levels and triggers a speaker alarm. If a gas leak is detected, the system emits a warning sound, alerting people nearby so they can respond promptly and safely.
